# CrewDock v0.1.1

Focused polish release for documentation, navigation, updates, and day-to-day workspace flow.

## Highlights

- Markdown files now support built-in preview and split view inside the editor
- Mermaid diagrams render directly in Markdown preview, with safer handling for common authoring mistakes
- Markdown preview gets a wider canvas so diagrams and richer docs are easier to read
- File explorer now shows Git-aware decorations so changed files stand out immediately
- CrewDock can check GitHub Releases and surface an in-app update prompt when a newer build is available
- Cmd/Ctrl+1 through Cmd/Ctrl+9 now jump directly between workspace tabs
- Terminal pane sizing and viewport behavior are tighter, with cleaner use of available workspace space
- Light theme status-bar contrast is improved so workspace, Files, Codex, Tasks, CPU, and memory labels stay readable

## Notes

- Downloads still ship through notarized macOS DMGs on GitHub Releases
- Update installation remains manual, but CrewDock can now notify you when a newer release is available
- This release continues to keep workspace state, drafts, layouts, and settings local to your machine
