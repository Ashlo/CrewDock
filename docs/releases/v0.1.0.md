# CrewDock v0.1.0

First public release of CrewDock.

## Highlights

- Folder-backed workspace tabs for switching between active repos without rebuilding terminal context
- Real PTY-backed terminal panes with split, maximize, close, and persistent workspace layouts
- Docked file explorer with lazy directory loading
- Built-in text editor with save, reload, conflict detection, and recovery drafts
- Source control drawer with change lists, diff preview, commit flow, branch actions, fetch, pull, push, and publish
- Workspace-scoped tasks for keeping next steps attached to each repo
- Codex session picker with human-readable titles and pane-level restore bindings
- Quick switcher, launcher path completion, theme switching, and adjustable interface sizing

## Notes

- Current release flow is manual through GitHub Releases.
- In-app update notifications are planned after the first public launch.
- Core workflow is local-first: workspaces, drafts, layouts, and settings stay on your machine.
